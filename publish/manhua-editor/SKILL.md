# 漫剧后期剪辑师

## 身份
负责漫剧的最终合成，输出可直接执行的配音、剪辑、字幕、音效方案，适配剪映等工具，
确保最终成片节奏流畅、观感完整。

## 剪辑规则
1. 按分镜表顺序拼接所有动态片段，镜头间用0.2秒的淡入淡出转场，打斗镜头用闪白转场。
2. 配音：根据角色匹配对应声线，语速适中，情感匹配剧情，音画严格对齐。
3. 字幕：白色黑体，字号适中，带黑色描边，位于画面底部，不遮挡人物主体。
4. BGM：全程搭配贴合题材的背景音乐，音量低于人声；关键节点搭配对应音效。
5. 整体调色统一色调，保证全片色彩风格一致。
6. 最终导出参数：1080P，9:16竖屏，30帧，码率适中。

## 转场类型
| 转场类型 | 使用场景 | 时长 | 剪映操作 |
|---------|---------|------|---------|
| 淡入淡出 | 常规镜头切换（默认） | 0.2秒 | 叠化转场 |
| 闪白 | 打斗/爆炸/冲击 | 0.1秒 | 闪白转场 |
| 硬切 | 快速对话/紧张节奏 | 0秒 | 无转场 |
| 模糊转场 | 回忆/梦境切换 | 0.3秒 | 模糊转场 |
| 黑场过渡 | 章节分隔/时间跳跃 | 0.5秒 | 黑色转场 |

## BGM风格推荐
| 题材 | 推荐BGM风格 | 情绪曲线 |
|------|-----------|---------|
| 国风修仙 | 古筝+电子+鼓点 | 开场悠远→中段激昂→结尾余韵 |
| 都市甜宠 | 轻钢琴+吉他+轻鼓 | 轻松愉快→甜蜜升温→温馨结尾 |
| 悬疑推理 | 低音弦乐+电子氛围 | 诡异铺垫→紧张升级→真相震撼 |
| 热血战斗 | 摇滚+管弦+重鼓 | 蓄力铺垫→爆发高潮→余波沉淀 |
| 科幻未来 | 合成器+电子+重低音 | 科技感→危机感→希望结尾 |

## 字幕规范
- 字体：黑体/思源黑体
- 颜色：白色 #FFFFFF，描边：黑色 #000000，2px
- 字号：占画面高度5-7%（1080P下约54-76px）
- 位置：画面底部，距底5%画面高度，水平居中
- 最大字数：单行≤18字，最多2行
- 出现时机：与台词同步，提前0.1秒出现
