## 🔧 工具链

### 图片生成（Stage 4 — Artist Agent）

| 优先级 | 模型 | 质量 | 分辨率 | 费用 | 条件 |
|:--:|------|:--:|------|:--:|------|
| 🥇 | **Seedream 直连** (火山引擎) | ⭐⭐⭐⭐⭐ | 原生2K | ¥0.15/张 | `VOLCENGINE_API_KEY` |
| 🥈 | **seedream** (Pollinations代理) | ⭐⭐⭐⭐⭐ | 原生2K | ~$0.03/张 | `POLLINATIONS_KEY` + Pollen余额 |
| 🥉 | **grok-imagine** | ⭐⭐⭐⭐ | 原生1080+ | ~$0.02/张 | Key + 余额 |
| 🏅 | **z-image-turbo** | ⭐⭐⭐⭐ | 原生1088×1920 | 免费 | Key |
| 🟢 | **旧API兜底** | ⭐⭐⭐ | ~768px→放大 | 免费 | 永远可用 |

```bash
node tools/generate_image.js \
  --prompt "Vertical portrait, Chinese anime ink wash style..." \
  --output "projects/项目名/images/镜头01.png" \
  --width 1080 --height 1920 \
  --project-dir "projects/项目名"
```

追踪日志：`projects/项目名/图片生成追踪.json`

### 配音合成（Stage 8 — Editor Agent）

| 优先级 | 后端 | 质量 | 费用 | 条件 |
|:--:|------|:--:|------|------|
| 🥇 | **火山引擎 TTS** | ⭐⭐⭐⭐⭐ 中文最自然·情感丰富 | 新用户送50万字符 | `VOLCENGINE_APP_ID` + `VOLCENGINE_ACCESS_KEY` |
| 🥈 | **Edge-TTS** | ⭐⭐⭐⭐ 稳定可靠 | 免费 | 无需配置 |

```bash
python3 tools/generate_tts.py --batch "projects/项目名/音频设计.json"
```

### 视频生成（Stage 5 — Animator Agent）

| 优先级 | 后端 | 质量 | 分辨率 | 条件 | 费用 |
|:--:|------|:--:|------|------|:--:|
| 🥇 | Seedance 2.0 | ⭐⭐⭐⭐⭐ | 720p | `SEEDANCE_API_KEY` + 余额 | $0.14-0.42/段 |
| 🥈 | Kling/可灵 | ⭐⭐⭐⭐ | 1080p | `KLING_API_TOKEN` | 每日免费66积分 |
| 🟢 | Ken Burns | ⭐⭐⭐ | 1080×1920 | 只需 OpenCV | 免费 |
| 🔄 | 静态视频 | ⭐⭐ | 1080×1920 | FFmpeg | 免费 |

```bash
python3 tools/generate_video.py \
  --image "projects/项目名/images/镜头01.png" \
  --motion "zoom_in" \
  --duration 5 \
  --prompt "camera slowly pushes in, cinematic" \
  --output "projects/项目名/videos/镜头01.mp4" \
  --backend auto \
  --project-dir "projects/项目名"
```

追踪日志：`projects/项目名/视频生成追踪.json`

## 🚀 快速开始

### 环境变量（按需配置）

```bash
# === 图片 + 配音（一个平台搞定）===
export VOLCENGINE_API_KEY="volc_xxx"         # 火山引擎 Ark → 🥇 Seedream 图片
export VOLCENGINE_APP_ID="app_xxx"           # 火山引擎 App ID → 🥇 TTS 配音
export VOLCENGINE_ACCESS_KEY="ak_xxx"        # 火山引擎 Access Key → 🥇 TTS 配音
# 注册地址: https://console.volcengine.com

# === 图片备选 ===
export POLLINATIONS_KEY="sk_xxx"             # Pollinations → 🥉🏅 付费/免费图片

# === 视频生成 ===
export SEEDANCE_API_KEY="sk_xxx"             # Seedance → 🥇 图生视频
# 注册地址: https://seedanceapi.org

export KLING_API_TOKEN="api-key-kling-xxx"   # Kling → 🥈 图生视频(每日免费)
# 注册地址: https://klingapi.com
```

## 📊 调用追踪

每次生成自动记录到项目目录下的 JSON 文件：

- `图片生成追踪.json` — 每张图用的后端、分辨率、时间
- `视频生成追踪.json` — 每段视频用的后端、费用、耗时

格式统一：
```json
{
  "generations": [
    {
      "shot": "镜头01",
      "backend": "z-image-turbo",
      "status": "success",
      "timestamp": "2026-07-07T12:00:00",
      "resolution": "1088x1920"
    }
  ]
}
```
